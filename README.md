SmartFit MongoDB Project - 
Health and wellness application to assist its users with the tasks such as 
logging their workouts, sports, meals, and personal metrics as well as the 
ability to find new places to work out at.

SETUP:
To set up and analyze the data, execute these commands in your mongosh terminal, in the order listed:

1. Setup Schema: Through create_collection.js

2. Populate Data: Through populate.js

3. Add Indexes: Through index.js

FILE OVERVIEW:
create_collection.js    Database Schema & Validation

populate.js             Data Seeding (Sample Data)

index.js                Performance Indexes

basic_CRUD.js           Basic Database Queries

aggregation.js          Analytical Aggregations

queriesWithExplain.js   Query Performance Tests

presentation/           Project Slides

documentation/          Final Report

sample_data/            Raw Sample Data
